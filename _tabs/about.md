## Apa?
Beberapa topik yang menjadi pemahaman penulis dan mungkin menjadi kerangka tulisan-tulisan dalam blog ini:
* **Seputar konsep dasar sinyal**, terutama audio digital dan **pemrosesannya** (DSP).
* **Pemrograman aplikasi *desktop*** menggunakan `C++`, `Python`, `Rust`, dll.
* ***Scripting desktop* dan web** menggunkan `Python` dan `JavaScript`.
* **Pemrograman *embedded*** menggunakan Arduino (`C++`) dan ESP-IDF (`C`).
* **Lebih detail tentang pemrograman** seperti tentang `CMake`, `mingw`, `Docker`, dll.
* **Hal-hal lain terkait *computer science*** seperti *open-source software*, *Linux*, *WAV*, dll.

## Bagaimana?
Blog ini ditulis dengan framework [Jekyll](https://jekyllrb.com/) dengan tema [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) yang di-*host* menggunakan [GitHub Pages](https://pages.github.com/).

<strong>Tools penting yang penulis gunakan:</strong>
* **Windows + [Windows Subsystem for Linux](https://ubuntu.com/wsl) (WSL)**: Untuk menjalankan berbagai macam distro Linux di Windows. Untuk mendapatkan *best of both worlds* (*user-frindly* Windows + *developer-friendly* Linux), test aplikasi *cross-platform*, isolasi *environment*, dsb. Blog ini sendiri ditulis di instansi Arch Linux WSL. *<u>Highly recommended</u>*.
* **[Visual Studio Code](https://code.visualstudio.com/)**: Text editor paling populer untuk pemrograman. Dengan bantuan banyak ekstensi-ekstensi yang tersedia di [marketplace](https://marketplace.visualstudio.com/vscode), fungsionalitasnya dapat disesuikan dengan bahasa, framework, hingga kebutuhan dan preferensi *user*. *<u>Highly recommended</u>*.
* **[Scoop](https://scoop.sh/)**: Alternatif `apt`, `pacman`, dsb. untuk Windows (meskipun lebih sebagai "*command line insaller*" dari pada "*package manager*"). *<u>Recommended</u>*.
* **[Windows Terminal](https://apps.microsoft.com/store/detail/windows-terminal/9N0DX20HK701?hl=en-id&gl=id&rtc=1)**: Terminal emulator terbaik untuk Windows, *built-in* sejak Windows 11. *<u>Recommended</u>*.
* **[Visual Studio](https://visualstudio.microsoft.com/) dan/atau *Build Tools*-nya**: IDE + *bundle compiler*, *SDK*, *Library* untuk pemrograman di Windows menggunakan `C++`, `C#`, dll. Cukup *<u>wajib</u>* jika ingin mengkompilasi aplikasi dengan *compiler* utama Windows (**Visual C++**).
